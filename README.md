# Image Editor (HTML, CSS, JavaScript)

![Screenshot](screenshot.png)

A browser-based image editor that applies real-time filters using pure HTML, CSS, and JavaScript — no libraries or frameworks.

## ✨ Features
- Upload and preview image
- Real-time filters: brightness, contrast, blur, grayscale, sepia, hue, invert, opacity
- Preset filters: Vintage, Dramatic, Cool, Warm, Black & White
- Reset and download edited image
- Responsive and clean UI

## 🧠 Concepts Used
- CSS `filter` property
- JavaScript DOM manipulation
- Event listeners
- Canvas API for image download

## 🚀 How to Run
Download the files and open `index.html` in your browser.

## 📌 Tech Stack
HTML • CSS • JavaScript
